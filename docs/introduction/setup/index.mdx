Configurar seu novo projeto de documentação é simples. A maioria das configurações essenciais do projeto pode ser definida dentro da pasta `settings`.

## Configurações

Esta seção fornece as configurações principais para seu site de documentação, como definir a URL do site, ícone e título.

```tsx showLineNumbers
export const url = ""                 // A URL para seu site de documentação
export const siteicon = ""            // Ícone exibido ao lado do nome do site no cabeçalho
export const sitename = ""            // Título do seu site de documentação
```

## Tags do Google

Esta seção permite que você integre o Google Tag Manager ao seu projeto de documentação. Você pode adicionar seu código GTM e configurar se está conectado.

```tsx showLineNumbers
export const gtm = ""                 // Adicione seu código do Google Tag Manager aqui
export const gtmconnected = true      // Defina como true se o GTM estiver conectado, caso contrário, defina como false
```

## Mecanismo de Busca

Configure as configurações essenciais de SEO para garantir que seu site de documentação esteja otimizado para mecanismos de busca. Isso inclui meta descrições, palavras-chave e detalhes de compartilhamento em redes sociais, como imagens e texto alternativo.

```tsx showLineNumbers
export const description = ""         // Descrição do seu site de documentação para SEO
export const keywords = ["", ""]      // Lista de palavras-chave para SEO do seu site de documentação
export const urlimage = ""            // A URL da imagem usada ao compartilhar nas redes sociais
export const imagealt = ""            // Texto alternativo para imagens compartilhadas, útil para leitores de tela
export const twitterhandle = ""       // Seu perfil do Twitter para compartilhamento social
```

## Marca no Rodapé

As configurações do rodapé permitem que você personalize as informações da empresa exibidas na parte inferior do site. Você pode adicionar o nome da empresa e a URL para a qual o nome será vinculado.

```tsx showLineNumbers
export const companyname = ""         // O nome da empresa exibido na seção de direitos autorais
export const companylink = ""         // A URL para a qual o nome da sua empresa deve ser vinculado
```

## Layout Geral

As configurações de layout controlam a aparência e a funcionalidade de vários elementos do site de documentação, como marca, barra lateral direita, opções de feedback, tabela de conteúdo e funcionalidade de rolar para o topo.

```tsx showLineNumbers
export const branding = true          // Defina como true para exibir nossos elementos de marca
export const rightsidebar = true      // Defina como true para habilitar a barra lateral direita com feedback, tabela de conteúdo e rolar para o topo
export const feedbackedit = true      // Controle para habilitar/desabilitar feedback no GitHub
export const tableofcontent = true    // Controle para habilitar/desabilitar a tabela de conteúdo gerada
export const totopscroll = true       // Controle para habilitar/desabilitar o botão de rolar para o topo
```

## Controle do GitHub

Se seus arquivos MDX estiverem hospedados no GitHub, você pode configurar o projeto para carregar documentos diretamente de lá. Esta opção pode ser útil para sincronizar atualizações de documentação diretamente com seu repositório.

```tsx showLineNumbers
export const loadfromgithub = false   // Defina como true para carregar seus documentos MDX diretamente do GitHub
```
